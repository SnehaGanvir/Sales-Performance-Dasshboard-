
# Amazon Store Sales Performance Dashboard

**Power BI Project**

## Overview

This project showcases an interactive Amazon Store Sales Performance Dashboard built using Power BI. The dashboard provides a unified view of sales, profit, orders, and product performance across regions, categories, and customer segments. It enables effective decision-making by transforming raw transactional data into clear, actionable insights.

## Key Highlights of the Dashboard

* Total **Sales: 2M**, **Profit: 175K**, **Orders: 5901**, and **Products: 1755** summarized at the top for quick reference.
* **Sales by Segment**, **Payment Mode**, and **Ship Mode** visualizations to understand customer behavior.
* A **Seasonal Sales Trend** line chart comparing year-wise performance (2019 vs 2020).
* **Regional analysis** using a map visual displaying sales distribution across states.
* **Category-wise sales** and **Top 3 profitable sub-categories** for product-level insights.
* **Quarter-wise Sales vs Profit** to identify growth patterns.
* **Return Status** highlighting order acceptance and returns.

## Tools and Components Used

* Power BI Desktop
* Power Query (Data Cleaning and Transformation)
* DAX Measures for KPIs and calculations
* Visuals used: Donut charts, bar charts, line charts, map visual, slicers, KPI cards

## Dataset

* Included sales transaction data with details on segment, category, ship mode, payment mode, profit, orders, and returns.
* Performed data cleaning and modeling before building the dashboard.

## Purpose

The dashboard is designed to:

* Identify sales trends across time and geography.
* Understand customer preferences and product performance.
* Track operational metrics such as returns and shipping modes.
* Provide leadership with insights for forecasting and strategy development.
